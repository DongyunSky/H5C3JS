[列表]

1. ul 标签

定义无序列表 只能包含 li 标签 可指定 onclick 属性

还有 id class style 属性

2. ol 标签

定义有序列表 同 ul 一样

除此之外还能指定3个属性

  1. start: 指定列表项的起始数字

  2. type: 指定使用哪种类型的编号(1-数字；A-数字；I-罗马数字。不推荐，推荐用css)

  3. reversed: 反转排序，目前没有任何浏览器支持该属性（红红火火恍恍惚惚）(明明支持)

3. li 标签

id style class onclick 属性

里面可包含 div 等多种内容

4. dl dt dd 标签

dl:

只能包含 dt dd 标签 其他和 li 差不多

dt:

定义标题列表项

只能包含文本、图像、超链接、文本格式化元素和表单元素

dd:

定义普通列表项

与 li 类似
